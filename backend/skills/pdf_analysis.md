# PDF Analysis Skill

## Overview
This skill covers PDF document analysis including reading, extracting text/tables, understanding document structure, and answering questions about PDF content.

## Capabilities
- Extract and analyze text from PDF documents
- Identify document structure (headers, sections, tables)
- Reference specific page numbers when answering questions
- Quote relevant passages using highlight markers
- Handle multi-page documents with cross-reference analysis

## Guidelines
- Always reference page numbers when citing specific content
- Use <<highlight>>quoted text<</highlight>> markers for text that should be highlighted in the PDF viewer
- When analyzing tables, preserve structure in your response
- For scanned/image PDFs, note that OCR quality may affect accuracy
- Provide comprehensive answers that draw from multiple sections when relevant

## Quick Reference
| Task | Approach |
|------|----------|
| Find specific info | Search across all pages, cite page numbers |
| Summarize | Read full document, identify key themes |
| Compare sections | Cross-reference multiple pages |
| Extract data | Look for tables, lists, and structured content |
